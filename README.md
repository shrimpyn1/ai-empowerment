# 思维的杠杆 | 智识时代的崛起

An interactive webpage exploring how AI is empowering non-technical professionals with unprecedented capabilities.

## Overview

This project visualizes the paradigm shift in the AI era, where critical thinking, communication, and cross-disciplinary skills are becoming more valuable than traditional technical expertise alone.

## Features

- **Interactive Statistics**: Animated counters and charts displaying real-world data
- **Smooth Animations**: Scroll-triggered reveals, parallax effects, and dynamic transitions
- **Terminal Simulation**: Interactive AI agent demonstration
- **Responsive Design**: Fully optimized for desktop and mobile devices
- **Data Visualizations**: Bar charts and donut charts showing skill value changes and AI adoption rates

## Deployment to GitHub Pages

### Step 1: Create Repository
1. Go to [github.com/new](https://github.com/new)
2. Name your repository (e.g., `ai-empowerment` or `ai-era`)
3. Set visibility to **Public**
4. Do NOT initialize with README (we already have one)
5. Click **Create repository**

### Step 2: Push Code
Run these commands in the project directory:

```bash
cd /Users/samchen/Documents/Coding\ Agent\ Space/ai-empowerment
git init
git add .
git commit -m "Initial commit: AI Empowerment interactive webpage"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git
git push -u origin main
```

### Step 3: Enable GitHub Pages
1. Go to your repository on GitHub
2. Click **Settings** → **Pages** (left sidebar)
3. Under "Source", select **Deploy from a branch**
4. Select **main** branch and **/ (root)** folder
5. Click **Save**
6. Wait 1-2 minutes for deployment
7. Your site will be live at: `https://YOUR_USERNAME.github.io/YOUR_REPO_NAME/`

## Local Development

Simply open `index.html` in any modern browser, or use a local server:

```bash
# Using Python
python -m http.server 8000

# Using Node.js (if npx available)
npx serve .
```

Then visit `http://localhost:8000`
